禁用触控板

sudo modprobe -r psmouse
启用：
sudo modprobe -a psmouse

永久禁用：
sudo vi /etc/modeprobe.d/blacklist.conf
blacklist psmouse

禁用触控板

禁用触控板：
synclient touchpadoff=1
启用触控板：
synclient touchpadoff=0
此时尽有触控板禁用，但trackpoint（小红帽）依然可用。
如果需要每次启动时都均用，那么可以在~/.bashrc中添加上述代码：
vim ~/.bashrc
在最后增加：synclient touchpadoff=1，保存退出

如果需要同时禁用touchpad和trackpoint可以使用：
sudo modprobe -r psmouse
启用为：sudo modeprobe -a psmouse
如果需要永久禁用，可以尝试：
sudo vi /etc/modprobe.d/backlist.conf
在其中添加以下行：
blacklist psmouse
此外，也可以在BIOS中禁用触控板和Trackpoint。

Vim拷贝系统剪切板

先检查是否支持clipboard:
vim –version | grep clipboard

如果显示+Clipboard等字样，表示支持这一功能，此时可以用以下命令操作：
“+y 代替y将选中的内容复制到系统剪切板，效果同ctrl+C
“+p 代替p将剪切板中的内容复制到指定位置，效果同ctrl+V

如果需要用y直接代替”+y复制到剪切板，可以在.vimrc中配置：
vim ./vimrc
增加如下行：
set clipboard=unnamed

如果版本中不支持Clipboard，即出现-Clipboard。
先安装需要的软件包：
sudo apt install vim-gtk

安装Sublime Text 3

Install the GPG key:
wget -qO - https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -
Ensure apt is set up to work with https sources:
sudo apt-get install apt-transport-https

Select the channel to use:
Stable
echo “deb https://download.sublimetext.com/ apt/stable/“ | sudo tee /etc/apt/sources.list.d/sublime-text.list

Dev
echo “deb https://download.sublimetext.com/ apt/dev/“ | sudo tee /etc/apt/sources.list.d/sublime-text.list

Update apt sources and install Sublime Text
sudo apt-get update
sudo apt-get install sublime-text

修改hosts文件

大多数Linux发行版中，hosts文件位于/etc/hosts。可以查阅一下已有的内容：
cat /etc/hosts

vim /etc/hosts

安装截屏软件flameshot

flameshot是一个非常不错的截屏软件，安装与使用也比较简单，功能也比较全。
安装方法：
sudo apt install flameshot
安装后软件的位置在/usr/bin/flameshot ，运行的参数是：/usr/bin/flameshot guibin/flameshot gui

可以创建一个系统快捷键，在Lubuntu启动菜单中是在”系统工具“/”setup hotkeys”中。在“Program”中添加一个新的项目，执行的命令是：/usr/bin/flameshot gui, 可以设置一个自己习惯的快捷键，例如：CTRL+ALT+A

安装Hexo

Hexo是一个静态站点生成程序，使用的开发技术是Nodejs。在Ubuntu及衍生版中的安装方法如下：

1、安装NodeJS
sudo apt install nodejs, npm
更换淘宝镜像：
sudo npm –global config set registry https://registry.npm.taobao.org
查看并检查配置：
npm config list

安装版本更新工具N：
sudo npm install n -g
更新版本，显示installed:版本号，表示更新成功。
sudo n stable

2、安装Hexo
sudo npm install -g hexo
