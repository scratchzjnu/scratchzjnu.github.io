Vim拷贝系统剪切板

先检查是否支持clipboard:
vim –version | grep clipboard

如果显示+Clipboard等字样，表示支持这一功能，此时可以用以下命令操作：
“+y 代替y将选中的内容复制到系统剪切板，效果同ctrl+C
“+p 代替p将剪切板中的内容复制到指定位置，效果同ctrl+V

如果需要用y直接代替”+y复制到剪切板，可以在.vimrc中配置：
vim ./vimrc
增加如下行：
set clipboard=unnamed

如果版本中不支持Clipboard，即出现-Clipboard。
先安装需要的软件包：
sudo apt install vim-gtk

修改hosts文件

大多数Linux发行版中，hosts文件位于/etc/hosts。可以查阅一下已有的内容：
cat /etc/hosts
典型输出如：

激活sublime text3 3211

此操作估计只针对3211有效，其它版本当然也可以试试。
1、修改hosts文件
vim /etc/hosts
在后面添加以下记录，建议在前面用”# 注释”的方式添加一个说明，以免时间久了不知道是用于什么的。

127.0.0.1 www.sublimetext.com
127.0.0.1 sublimetext.com
127.0.0.1 sublimehq.com
127.0.0.1 license.sublimehq.com
127.0.0.1 45.55.255.55
127.0.0.1 45.55.41.223
0.0.0.0 license.sublimehq.com
0.0.0.0 45.55.255.55
0.0.0.0 45.55.41.223

保存并退出vim。
